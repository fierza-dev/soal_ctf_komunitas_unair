Metode 1: Mengunduh ZIP (tanpa Git)

    Buka Repositori GitHub
        Akses halaman repositori di GitHub (contoh: https://github.com/username/repository-name).

    Klik Tombol "Code"
        Di halaman utama repositori, cari tombol berlabel Code di dekat kanan atas daftar file.

    Pilih "Download ZIP"
        Setelah klik tombol "Code", akan muncul opsi dropdown. Klik Download ZIP untuk mengunduh seluruh repositori sebagai file ZIP.

    Ekstrak File ZIP
        Setelah unduhan selesai, buka file ZIP menggunakan aplikasi pengarsipan seperti WinRAR, 7-Zip, atau alat bawaan sistem operasi. Ekstrak isinya ke folder pilihan Anda.

Metode 2: Clone Repositori dengan Git

Menggunakan Git memberi Anda kemampuan untuk berinteraksi lebih dinamis dengan repositori.

    Install Git (Jika Belum Terinstal)
        Windows: Unduh installer dari Git-SCM dan ikuti proses instalasi.
        Linux: Gunakan perintah seperti sudo apt install git (Ubuntu) atau sudo yum install git (CentOS).
        MacOS: Gunakan Homebrew: brew install git.

    Salin URL Repositori
        Buka halaman repositori di GitHub dan klik tombol Code, lalu salin URL di bawah bagian HTTPS (contoh: https://github.com/username/repository-name.git).

    Clone Repositori
        Buka terminal atau Command Prompt, navigasikan ke folder tempat Anda ingin menyimpan repositori menggunakan perintah cd path/to/folder.
        Jalankan perintah:

        git clone https://github.com/username/repository-name.git

        Git akan mengunduh semua file repositori ke folder baru.

Metode 3: Mengunduh File Tunggal

Jika Anda hanya membutuhkan satu file, Anda tidak perlu mengunduh seluruh repositori.

    Pilih File
        Buka file yang ingin Anda unduh di repositori GitHub.

    Klik Tombol "Raw"
        Setelah file terbuka, klik tombol Raw di kanan atas.

    Simpan File
        File akan ditampilkan dalam format mentah di browser. Klik kanan dan pilih Save As (Simpan Sebagai) untuk menyimpannya di komputer Anda.
